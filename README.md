# JavaScript Projects

These are some projects I made using JavaScript.

## Projects

- One-Page Website
- Portfolio Site


## One-Page Website

This project is an automobile-themed one-page website called **Auto World**. It includes a navigation bar, home section, image gallery, embedded video, contact form, and footer.

This project was upgraded with JavaScript to include a lightbox modal image gallery. Users can click on thumbnail images to view larger versions of the images in a popup modal. The gallery also includes previous and next controls.

## Portfolio Site

This project is a personal portfolio website created to showcase my background, skills, and projects. It includes a home section with a video background, an About section, a GitHub section, a slideshow, a popup contact form, and a footer.

The portfolio was upgraded with JavaScript features, including slideshow controls, clickable dots, a popup contact form, and form validation.
